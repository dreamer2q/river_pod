安装 [Riverpod]， 然后编写一个 “Hell world” 示例。

## 需要安装什么库

在你开始之前，你需要意识到 [Riverpod] 被分成多个库，不同的库有些许轻微的差异。
你需要安装那一个 [Riverpod] 库，取决于你的应用类型。
你可以参考下面的表格，来了解你需要安装哪一个库。

| 应用类型                  | 库名                                                                               | 描述                                                          |
| ------------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| 只有 Flutter              | [flutter_riverpod]                                                                 | Flutter 使用 [Riverpod] 的基础用法。                          |
| Flutter + [flutter_hooks] | [hooks_riverpod]                                                                   | 允许 [flutter_hooks] 和 [Riverpod] 同时使用.                  |
| 只有 Dart (无 Flutter)    | [riverpod](https://github.com/rrousselGit/river_pod/tree/master/packages/riverpod) | 一个特定版本的 [Riverpod] 库，移除了所有和 Flutter 相关的类。 |

## 安装库

当你了解需要使用那些库时，就可以添加下面的内容到你的 `pubspec.yaml` 中：

<Tabs
  groupId="riverpod"
  defaultValue="hooks_riverpod"
  values={[
    { label: 'Flutter + flutter_hooks', value: 'hooks_riverpod', },
    { label: '仅 Flutter', value: 'flutter_riverpod', },
    { label: '仅 Dart', value: 'riverpod', },
  ]}
>
<TabItem value="hooks_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_hooks: ^0.18.0
  hooks_riverpod: ^2.0.0-dev.4
```

然后运行 `flutter pub get`。

</TabItem>
<TabItem value="flutter_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.0.0-dev.4
```

然后运行 `flutter pub get`。

</TabItem>
<TabItem value="riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0-0 <3.0.0"

dependencies:
  riverpod: ^2.0.0-dev.4
```

然后运行 `dart pub get`。

</TabItem>
</Tabs>

就是这样。你已经将 [Riverpod] 添加到你的 App 中了！
